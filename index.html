CSS

Cascading Style Sheets (Folha de Estilos em Cascata), é uma linguaggem usada para definir a aparência visual de um site.

CSS não é uma imagem de programação. É uma linguagem declarativa de estilos.

O CSS consegue definir:

  • Cores
  • Fontes
  • Espaçamentos
  • Controle de Layout
  • Responsividade
  • Animações
  • Efeitos visuais
  • Posicionamento de elementos

Formas de trabalhar com o CSS

Existem 3 formas clássicas de se trabalhar com o CSS:

  • Inling
  • interno
  • Externo
